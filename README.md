Introdução

O KantSpeak é um sistema educacional desenvolvido com o objetivo de auxiliar no ensino de idiomas, com foco especial em crianças, incluindo aquelas com Transtorno do Espectro Autista (TEA). A proposta central do projeto é utilizar recursos interativos, visuais e tecnológicos para tornar o processo de aprendizagem mais acessível, inclusivo e eficiente.

O sistema foi concebido dentro de uma perspectiva pedagógica que valoriza a mediação tecnológica como ferramenta de apoio ao ensino, promovendo engajamento, autonomia e desenvolvimento cognitivo.
